# 🌍 World Weather App 🌤️

A modern weather application to check real-time weather of any city worldwide.

---

## 🚀 Live Demo

👉 https://world-weather-free.vercel.app/

---

## ✨ Features

* 🌍 Search any city or country
* 🌡️ Temperature & feels like
* 🕒 Live time and date
* 🔐 Secure API

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Node.js
* OpenWeather API

---

## ⚙️ How It Works

1. User inputs city name
2. Request sent to `/api/weather`
3. Serverless function fetches data
4. Clean response sent to frontend
5. UI updates instantly
